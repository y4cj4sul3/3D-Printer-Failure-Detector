How does it work?

The following diagram is the pipeline of our system.
By comparing the segmented image from live camera and the projected image from 3D model, we can detect failure if there exists major difference.

  • Monitoring

    Use the 3D printer built-in live camera, and capture the raw image data frame by frame from the stream.

  • Calibration

    Calibrate the raw image to remove of lens distortion.

  • Segmentation

    Segment the printing object for later processing.

  • 3D Model

    Reconstruct the 3D model from gcode format.

  • Projection

    Project 3D model to the same view of the live camera.

  • Difference

    Compare two processed images, calculate the pixel difference or the motion vector, threshold the value for the result.
